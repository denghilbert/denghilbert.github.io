Overview

We leverage a 3D foundation model (e.g., VGGT) to learn geometrically aligned features that guide Gaussian primitive prediction and pose refinement via bundle adjustment, significantly improving final rendering quality.

Method

Using a pretrained VGGT backbone, we align DPT adapter features by reprojecting query points via predicted depth and camera parameters, enforcing similarity between corresponding source and target locations.
